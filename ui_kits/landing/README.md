# Landing Page UI Kit

High-fidelity recreation of the Donjons & Jambons one-page guild website.

## Sections covered
1. **Navbar** — Fixed, blur backdrop, logo + links + Discord CTA + mobile burger
2. **Hero** — Full-viewport, banner image, floating logo, CTA buttons
3. **About** — Two-column text + crest illustration
4. **Activities** — 6-card grid with SVG icons
5. **Members** — Filterable grid by role (Tank/Heal/DPS)
6. **Events** — Cards with slot progress bars
7. **Gallery** — Masonry + lightbox
8. **FAQ** — Accordion
9. **Join** — CTA + guild rank timeline
10. **Footer** — Brand + nav + Discord

## Source
- GitHub: https://github.com/Jungso-GB/donjons-jambons
- `index.html` — production HTML (vanilla JS/CSS)
- `assets/style.css` — full stylesheet

## Components
All components are vanilla HTML/CSS/JS. No framework dependencies.
